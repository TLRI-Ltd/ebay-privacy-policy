Privacy Policy for Vilbury eBay Integration

Last Updated: January 16, 2026

Application Name: Vilbury eBay Integration

Developer: TRLI Ltd

Contact: marius@vilburylondon.com

1. Introduction

This privacy policy explains how TRLI Ltd ("we", "us", "our") collects, uses, and protects data when using our eBay integration application. This application is used exclusively for internal business operations at Vilbury London.

2. Application Purpose

This application is an internal business tool used to:

  • Manage inventory listings across eBay marketplaces
  • Process and fulfill customer orders
  • Maintain accurate product information and pricing
  • Manage business account settings and compliance (e.g., VAT)
Important: This application is NOT available to external users or third parties. It is used solely by authorized TRLI Ltd personnel for business operations.

3. Data Collection

3.1 What Data We Access

Through eBay's API, our application accesses:

  • Inventory Data: Product listings, SKUs, titles, descriptions, pricing, quantity, images
  • Order Data: Order details, customer shipping information, order status, transaction records
  • Account Settings: VAT configuration, business seller status, marketplace settings, account preferences

3.2 Authentication Data

We store:

  • OAuth refresh tokens (encrypted, used to authenticate with eBay API)
  • Access tokens (temporary, auto-refreshed every 2 hours)

4. How We Use Data

Data accessed through eBay API is used to:

  • Inventory Management: Synchronize product listings, update pricing, manage stock levels
  • Order Fulfillment: Process customer orders, print shipping labels, track deliveries
  • Business Analytics: Generate reports on sales performance and inventory turnover
  • Compliance: Ensure accurate VAT settings and regulatory compliance across marketplaces

5. Data Storage and Security

5.1 Where Data Is Stored

  • Primary Database: Supabase (PostgreSQL) - cloud-hosted with encryption at rest
  • Server Location: Data centers in the European Union (EU)
  • Local Cache: Temporary token cache files on secure company servers

5.2 Security Measures

  • All API communications use HTTPS/TLS encryption
  • OAuth 2.0 authentication with rotating access tokens
  • Database access restricted to authorized personnel only
  • Regular security updates and monitoring
  • Access tokens expire every 2 hours
  • Refresh tokens expire every 18 months

6. Data Sharing

We do not share, sell, or rent any eBay data to third parties.

Data access is strictly limited to:

  • Authorized TRLI Ltd employees who require access for business operations
  • Supabase (our database provider) - covered under their data processing agreement

7. Data Retention

  • Inventory Data: Retained while listings are active, plus 12 months for historical records
  • Order Data: Retained for 7 years to comply with UK tax and accounting regulations
  • Authentication Tokens: Access tokens deleted after expiry (2 hours); refresh tokens retained until replaced

8. Your Rights (eBay Sellers)

As this is an internal application used on TRLI Ltd's own eBay seller accounts, standard GDPR rights apply:

  • Access: Request copies of data we hold
  • Rectification: Request correction of inaccurate data
  • Erasure: Request deletion of data (subject to legal retention requirements)
  • Data Portability: Request data in machine-readable format

To exercise these rights, contact: marius@vilburylondon.com

9. Customer Data (eBay Buyers)

When processing eBay orders, we receive customer information (names, addresses, email) from eBay. This data is:

  • Used solely for order fulfillment (shipping, customer service)
  • Subject to eBay's own privacy policy and data processing terms
  • Retained per UK tax regulations (7 years)
  • Protected with the same security measures as other business data

10. Compliance

This application complies with:

  • UK GDPR (General Data Protection Regulation)
  • eBay Developer Program Policies
  • UK Data Protection Act 2018

11. Changes to This Policy

We may update this privacy policy periodically. Changes will be posted with a new "Last Updated" date. Continued use of the application after changes constitutes acceptance of the updated policy.

13. eBay Developer Program Compliance

This privacy policy is provided in compliance with eBay Developer Program requirements. Our application:

  • Uses OAuth 2.0 for secure authentication
  • Requests only the minimum necessary permissions (scopes)
  • Implements industry-standard security practices
  • Complies with eBay API rate limits and usage policies

Application Scopes Requested:

  • https://api.ebay.com/oauth/api_scope/sell.inventory.readonly (read inventory)
  • https://api.ebay.com/oauth/api_scope/sell.inventory (manage inventory)
  • https://api.ebay.com/oauth/api_scope/sell.fulfillment (manage orders)

This privacy policy applies exclusively to TRLI Ltd's internal use of eBay API for business operations.
